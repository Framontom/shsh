# 文档概述

本文档是2024春季学期软件工程课程的项目文档。

本文档内容：

- 用户操作文档：平台各个部分的操作说明及注意事项
- 需求分析：各类用户在平台上所要进行的操作及操作流程图
- 模块设计：系统的整体架构
- API文档：前后端对接过程中所使用的API的规范
- 模型文档：后端存储各种数据的格式
- 技术选型与部署: 介绍工程选择的技术和框架以及部署情况
- 测试文档：单元测试覆盖到的情况列表
